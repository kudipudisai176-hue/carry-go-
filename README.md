# CarryGo Logistics - East Godavari

Modern, reliable, and fast parcel logistics platform connecting Senders with local Travellers.

## Project info
CarryGo is a localized logistics solution designed for safe parcel transport across East Godavari district.

## Key Features
- **Sender Dashboard**: Easy parcel creation with image compression.
- **Traveller Dashboard**: Real-time delivery tracking and acceptances.
- **Security**: OTP-based verification for pickup and delivery.
- **Payment**: Integrated wallet and escrow system.

## Getting Started
Follow these steps:

```sh
# Step 1: Install dependencies
npm i

# Step 2: Start the frontend development server
npm run dev

# Step 3: Start the backend server (optional)
cd backend
npm run dev
```

## Technologies
This project is built with:
- **Frontend**: Vite, React, TypeScript, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, Supabase (Postgres & Auth)
- **UI Components**: shadcn-ui, Lucide Icons
